# spambuster
a spam detector 


# Download data
```sh
bash data.sh
```

# Setup the Project
```sh
python -m venv env
source ./env/bin/activate
pip install -r requirements.txt
```

# Usage(TESTING)
```sh
source env/bin/activate
make
```

# Usage(General)
```sh
source env/bin/activate
./app -i <INPUT PATH TO THE TEXT FILE>
```
